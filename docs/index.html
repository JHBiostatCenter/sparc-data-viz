Welcome

This site contains materials for learning data visualization in R using ggplot2 and the grammar of graphics approach. These materials emphasize Edward Tufte’s principles of graphical excellence: well-designed presentations of interesting data that tell the truth.

## Welcome

This site contains materials for learning data visualization in R using `ggplot2` and the grammar of graphics approach. These materials emphasize Edward Tufte's principles of graphical excellence: well-designed presentations of interesting data that tell the truth.
